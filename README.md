# KeenKeeper 👥

A friendship management web app that helps users keep track of their relationships, contact goals, and interaction history.

---

## Project Description

KeenKeeper is a responsive friendship tracker application where users can manage their friends, monitor contact goals, and keep a history of calls, texts, and video interactions.

The app helps users stay connected with important people in their lives by showing when a friend is overdue for contact, almost due, or on-track.

---

## Key Features

* Responsive design for mobile, tablet, and desktop devices
* Friend cards with profile picture, tags, status, and contact information
* Detailed friend profile page with stats, goals, and quick check-in actions
* Timeline page to track all Call, Text, and Video interactions
* Analytics page with Pie Chart visualization using Recharts
* Filter options for timeline interactions
* Loading spinner while fetching friend data
* Toast notifications for user actions
* Custom 404 page for invalid routes
* Page reload support after deployment
